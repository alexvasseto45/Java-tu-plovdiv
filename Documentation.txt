Конзолно Java приложение за управление на библиотечна система, свързано с MongoDB. Съществуват два типа юзъри 
(администратор и обикновен юзър). В монго са създадени 5 колекции.
Функционалности:
Управление на библиотеки
	- създаване на библиотеки

Регистрация/вход на потребители
	- регистриране на потребител и избиране на неговия тип admin/user
	- вход с потребителско име и парола

Разглеждане на книги
	- разглеждане на книги за профил
	- рабглеждане на книги за библиотека
		- показване на среден рейтинг от всички рейтинги за тази книга
		- опция за добавяне на книги към акаунт
	- добавяне на книги в библиотека(за администратори)
	- премахване на книги от библиотека (за администратори)
	- логаут

Оценяване на книги
	- в менюто с книги за юзър те са selectable
	- избиране на книга и показване на меню
	- опция за рейтинг на книга
	- опция за премахване на книга

Добавяне/премахване на книги (за администратори)
	- добавя книга по име
	- премахва книга по име


Колекции, които се използват 
usersCollection:
	{	
	"_id": ObjectId()
	"username": string
	"password": string
	"library_id" string
	"role" string (admin/user)
	}
librariesCollection
	{	
	"_id": ObjectId()
	"name": string
	}
bookRatingsCollection
	{	
	"_id": ObjectId()
	"book_name": string
	"from_user": string
	"book_id": ObjectId()
	"rating": int
	}
userBooksCollection
	{	
	"_id": ObjectId()
	"book_id": ObjectId()
	"owner": string
	"rating": int
	}
booksCollection
	{	
	"_id": ObjectId()
	"title": string
	"library_id": string
	}
